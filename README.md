# A Yekyll based Github pages

Gulp based build system with:

    - browser-sync
    - gulp
    - gulp-autoprefixer
    - gulp-concat
    - gulp-include
    - gulp-sass
    - gulp-sourcemaps
    - gulp-uglify
    - node-sass
		
		
## Installation
npm install
bower install susy --save
gem install
bundle install
bundle exec gulp
